 * Checks if the input value is Number.

 * @name isString
 * @since v0.0.6
 * @module Type
 * @description * -> Boolean

 * @return {Boolean} `true` if `x` is `String`, otherwise `false`.
 * @example
 *
 * isString('lol') //=> true
 * isString(5) //=> false
